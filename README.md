# Docker DEMOS

## PWD Session erzeugen
```
open http://play-with-docker.com
```

## Dateien kopieren
```
git clone https://github.com/git-mgeo/dockerdemos.git
```


## Demos
* Builderdemo
* Fotos von der Dockercon (kein Build notwendig)

```

make run
```
